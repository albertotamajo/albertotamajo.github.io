2021-07 - 2021-09
Deep Learning Research Internship
Mainz University of Applied Sciences, Mainz, Germany
Advisor: Dr. Bastian Pla
Keywords: Computer Vision, Deep Learning, Geometric Deep Learning
Conducted literature research on Geometric Deep Learning for point cloud
encoding
Conducted literature research on Generative Adversarial Networks and
Variational AutoEncoders for point cloud generation
Proposed a novel graph convolutional neural network and adjacency matrix
operation (Shrinking Layer) capable of exploring self-correlations, local-
correlations and non-local correlations between points for the encoding of
point clouds. The Shrinking Layer is a permutation invariant point cloud feature
extractor which employs the whole array of CNNs basic ideas: local receptive
fields, shared weights, and pooling. Furthermore, it is a fully differentiable neural
network module that can be plugged into existing deep neural network models
to improve their performance
Proposed a novel graph convolution neural network and adjacency matrix
operation (Stack Shrinking Layer), based on Shrinking Layer, capable of
extracting multiple point cloud features simultaneously in the same fashion as
CNNs extract multiple features on 2D images. Thus, the Stack Extending Layer
fully resembles current CNNs architectures
Implemented the proposed architectures and their training procedures using
the Pytorch and Pytorch Geometric libraries. The implementation supports multi-
GPU utilisation on cluster computing
Employed the Stack Extending Layer in a neural network architecture, dubbed
ShrinkingNet, for point cloud classification tasks. The architecture achieved an
outstanding 91.25% accuracy on the ModelNet10 dataset. This work might result
in a publication together with Dr. Bastian Pla
